PUBLICAÇÃO V13 — UX ÁGIL

1. Apps Script
- Abra o Apps Script ligado à planilha-base atual.
- Substitua o conteúdo completo de Code.gs pelo arquivo backend/Code.gs deste pacote.
- Salve o projeto.
- Em Implantar > Gerenciar implantações, abra a implantação Web App atual e publique uma Nova versão.
- Mantenha Executar como: você; Quem tem acesso: qualquer pessoa.
- Preserve a URL /exec já configurada em docs/js/config.js.

2. Sincronização administrativa
- Execute ativarSincronizacaoAdministrativa() uma vez apenas se o gatilho ainda não estiver ativo.
- Alterações diretas em Cadastros, Palpites ou Resultados limpam o cache e aparecem no site automaticamente.

3. GitHub Pages
- Substitua o conteúdo publicado pela pasta docs inteira.
- Os arquivos JavaScript já usam ?v=13; publique todos para que o navegador carregue a versão nova.

4. Teste após publicar
- Cadastre uma pessoa nova e confirme que o formulário de palpites abre sem pop-up bloqueante.
- Exclua a pessoa e confirme que a tela retorna ao formulário com aviso discreto.
- Apague um registro direto na planilha e valide que a sessão desaparece automaticamente no site.
